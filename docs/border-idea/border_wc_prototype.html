Vanilla Breeze experiment

Border effects as a high-touch UI skin.

This gallery treats borders as progressive enhancement: semantic content inside, CSS tokens around it, and optional JavaScript for reactive behavior. Some effects are practical. Some are intentionally weird to test the boundaries of <border-wc>.

CSS gradients custom properties reduced-motion aware JS reactive layer

Live controls

Shuffle the theme tokens or change the hero border effect. The content stays ordinary HTML.

Future renderer idea: CSS for most cases, canvas/WebGL for shader borders, particle edges, fluid noise, and image-based masks.

Design implication

The component does not need to own layout or content semantics. It can be a very small decoration primitive that composes with Vanilla Breeze tokens. That makes it useful for cards, callouts, status boxes, labs, games, dashboards, drop zones, and marketing sections.
