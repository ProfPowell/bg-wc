Ship the feeling, not just the feature.

A grain layer over everything, conic spotlight borders on the cards, and a hero whose mesh-gradient warms up as you scroll — every effect is one <bg-wc>.

Three effects, zero libraries

Each card's animated border is a conic preset behind an opaque inset panel.

Grain overlay

A fixed preset="grain" with mix-blend-mode: soft-light textures the whole page like film stock. Pointer-events off, so it never gets in the way.

Spotlight borders

A rotating conic bg-wc sits behind a panel inset by 1.5px — only the rim shows, so the border sweeps with light. No @property needed.

Scroll-linked

The hero reads --bg-wc-intensity every frame, so a scroll handler that sets that variable (and a hue-rotate) makes the background respond as you move.

Grain · conic · scroll-linked params — all from @profpowell/bg-wc.
