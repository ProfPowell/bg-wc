<gl-wc>

Theme-aware graphics-layer web component

Themed demos → GitHub →

Animated backgrounds for any HTML

Nine presets across WebGL and Canvas2D. Reads your design tokens via shadow-DOM inheritance. Auto-pauses off-screen and on tab hide.

<gl-wc preset="mesh-gradient" intensity="0.65">…</gl-wc>

Preset gallery

Quick start

<script type="module" src="./src/gl-wc.js"></script>

<gl-wc preset="dither" intensity="0.7">
  <h1>Hero headline</h1>
  <img slot="fallback" src="hero.webp" alt="">
</gl-wc>

Tokens consumed: --color-background, --color-foreground, --color-primary, --color-accent, --color-info, --color-success, --color-warning, --color-error. Override any with --gl-wc-color-*.
